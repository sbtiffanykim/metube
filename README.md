# METUBE

### Routers

/ -> Home
/join -> Sign up
/login -> Login
/search -> Search

/users/:id -> See User
/users/edit -> Edit MY profile
/users/delete -> Delete My profile
/users/logout -> Logout

/videos/:id -> See a Video
/videos/:id/edit -> Edit a Video
/videos/:id/delete -> Delete a Video
/videos/upload -> Upload a Video
